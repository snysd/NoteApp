# NoteAppGuide

このアプリはメモ帳アプリです。  
メモの内容を保存します。
・テキストの編集スペースを用意、保存可能。削除できる。
・作成したテキストをほかのユーザーと共有できる。
・DBに保存し、ダウンロードできる。ダウンロードしたファイルは任意の場所に保存できる。
